# solo-french-horn-opus-48

Solo French horn samples — sustained tones (blowing, signal) + short articulations (attack, staccato). All recorded by PhonosUPF (CC0). Articulation-based samples at the instrument's natural pitch; rootNote set to C4 (MIDI 60) as default.

## About

This repo holds **48 kbps Opus previews** for fast streaming in the web DAW.
Opus at 48k delivers near-CD quality at ~35% the size of MP3 128k.

- **Library**: Solo French Horn
- **Files**: 10 Opus previews
- **Bitrate**: 48 kbps VBR
- **Container**: Ogg Opus (.opus)
- **License**: CC0-1.0
- **Source**: https://freesound.org/people/PhonosUPF/
- **Branch**: `main`
